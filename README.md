# pay-qr

pay-qr is CLI tool for generate QR codes for payments in Slovakia.

## Installation

Use npm for instalation:

```bash
npm install -g pay-qr
```

## Usage
Run command:
```bash
pay-qr
```
Then follow the instructions. It is easy and user-friendly!
